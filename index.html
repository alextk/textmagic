textmagic gem

is a Ruby interface to the TextMagic's Bulk SMS Gateway.

Snippet:

api = TextMagic::API.new(username, password)
api.send 'Hi Wilma!', '999314159265'
api.receive
# => ['999314159265: Hi Fred!']
